# ProxySwitch 项目说明

## 项目定位

ProxySwitch 是一个独立的 Tauri 桌面工具，用于在界面中管理代理开关、状态、自启和连通性诊断。

## 项目来源

- 基座来源：/Users/houshangjun/project/switch
- 当前主开发目录：/Users/houshangjun/project/proxyswitch
- 远端仓库：git@github.com:zzktz/proxyswitch.git

## 目录约定

- docs/：只放中文简体文档
- tmp/：放历史资料、临时保留文件、迁移过程中不再直接使用的内容

## 当前目标

- 一键开启 / 关闭代理
- 显示当前状态
- 管理开机自启
- 提供 GitHub 连通性诊断
